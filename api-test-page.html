  • Environment: Development

📝 Usage Flow

  1. Register: Create a new user account using the register endpoint
  2. Confirm Email: Check your email and confirm your account (if required)
  3. Login: Get a JWT token using the login endpoint
  4. Generate Recipe: Use the JWT token to call the recipe generation endpoint
Note: The recipe generation endpoint requires authentication. You must first register/login to get a JWT token, then include it in the Authorization header as "Bearer YOUR_TOKEN".
